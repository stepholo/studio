# **App Name**: The Circle

## Core Features:

- Unified Credit Score: Aggregates financial activity from various institutions to compute a unified credit score.
- Financial Institution Connection: Allows users to connect various financial accounts (M-Pesa, Banks, SACCOs, etc.) to build their score.
- Transaction Aggregation: Aggregates transactions from connected institutions into a single view, categorized by income, spending, loans and savings.
- Insights Generation: Provides insights into user spending habits, repayment behavior, and saving streaks to help improve credit score.
- Personalized Credit Tips: Offers tailored tips to improve credit score based on individual financial behavior, provided as tool-assisted suggestions.
- Secure Onboarding: Guides new users through a secure signup and onboarding process, requesting necessary permissions.
- Automated Statement Upload: Enables users to automatically upload statements and sync data from various institutions for seamless data aggregation.
